micro framework

Why ?

  • easy for beginers, powerfull for profesionals, only 111 definitions, building blocks for any design system
  • dead simple html semantics, mobile, grid, printing support out of box
  • build for code generated content React, Vue, Angular, .NET, php
  • safe to include in your styles, friendly to other frameworks (just remove definitions 001 002 003 004)
  • Support for night mode
